Welcome to ClassMaker!

This library is for creating classes at runtime. ClassMaker takes dynamic EPackage, invokes generator (currently Ecore standart, but contributions are welcome), then PDE exporter, installs result into host and supplies loaded into its own OSGi runtime, EPackage class to client code.
